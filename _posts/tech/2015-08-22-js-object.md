# 继承的几种方式
## 借用构造函数实现继承
```javascript
function Parent() {
    this.name = 'parent';
}
function Child() {
    Parent.call(this);
    this.type = 'child';
}
var child = new Child();
```
> Child1无法继承Parent1的原型对象，并没有真正的实现继承（部分继承）

## 借用原型链实现继承
```javascript
function Parent1() {
    this.name = 'parent1';
}
function Child1() {
    this.type = 'child1'
}
Child1.prototype = new Parent1();
```
> 原型对象的属性是共享的。

### 组合式继承
```javascript
function Parent2() {
    this.name = 'parent2';
}
function Child2() {
    Parent2.call(this);
    this.type = 'child2';
}
Child2.prototype = Object.create(Parent2.prototype);
Child2.prototype.constructor = Child2;
```
